Checkers Game

To run: 
  1. open command line for the game's folder
  2. run "g++ checkers.cc space.cc game.cc main.cc"
  3. run "./a.out" or just "a.out"

To play:
  Enter the row and column number of the piece you want to move,
  then with no spaces enter the row and column number of the 
  place you want to move to. An example starting move:
  
  3241

  Movies your Piece at row 3, column 2 to row 4, column 1. (diagonal forward)
